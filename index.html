ฟังก์ชันครอบคลุมทุกงานบริหารพัสดุโรงเรียน

รวดเร็ว แม่นยำ และทำงานร่วมกันได้แบบไร้รอยต่อทั้งผู้บริหาร ครู และเจ้าหน้าที่พัสดุ

1. จัดการพัสดุ (วัสดุสิ้นเปลือง)

เบิก-จ่ายสต็อก ตัดสต็อกและเตือนวัสดุใกล้หมดแบบเรียลไทม์ พร้อมบันทึกประวัติการรับเข้าและคืนวัสดุอัตโนมัติ

2. ทะเบียนและตรวจสอบครุภัณฑ์

ค้นหาและพิมพ์สติ๊กเกอร์ QR Code / Barcode รองรับการตรวจสอบครุภัณฑ์ผ่านมือถือเพียงแค่เดินสแกนแล้วเลือกสถานะ

3. แจ้งซ่อมบำรุงและจัดซื้อจัดจ้าง

ครูผู้สอนสามารถสแกนแจ้งซ่อมครุภัณฑ์พร้อมถ่ายภาพแนบได้ทันที ระบบจัดซื้ออัปเดตยอดสต็อกอัตโนมัติเมื่อตรวจรับ

ระบบสำรองและจัดการข้อมูล (Database Backup & Restore)

ผู้ดูแลระบบสามารถส่งออกข้อมูลทั้งหมดเก็บไว้ในรูปแบบไฟล์ หรือนำเข้าไฟล์ข้อมูลเดิม และล้างประวัติธุรกรรมเพื่อป้องกันฐานข้อมูลเต็มได้ที่นี่

ส่งออกข้อมูลระบบ (Backup Database)

ดาวน์โหลดตารางข้อมูลทั้งหมดในระบบคลังพัสดุ ครุภัณฑ์ การแจ้งซ่อม และรายชื่อผู้ใช้งานเก็บเป็นไฟล์ JSON ไว้ในเครื่องคอมพิวเตอร์ของคุณอย่างปลอดภัย

นำเข้าข้อมูลสำรอง (Restore Database)

เลือกไฟล์สำรองข้อมูล JSON ที่เคยดาวน์โหลดไว้เพื่ออัปเดตเข้าระบบ ข้อมูลที่มี UUID ตรงกันจะได้รับการอัปเดตแบบอัตโนมัติ (Upsert) เพื่อความปลอดภัย

ล้างข้อมูลและประวัติธุรกรรมเก่า (Purge Old Transaction Data)

เพื่อความสบายใจในการใช้งานเซิร์ฟเวอร์ฟรี (Supabase 500MB Limit) คุณสามารถลบประวัติพัสดุและรายการซ่อมบำรุงที่บันทึกไว้นานแล้วออกไปได้ โดยระบุขอบเขตวันเวลา (แนะนำให้ทำการสำรองข้อมูลก่อนรันคำสั่งนี้)
